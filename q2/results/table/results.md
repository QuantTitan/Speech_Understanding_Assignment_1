# Results — Q2 Speaker Disentanglement

## Evaluation Set: LibriSpeech test-clean (synthetic trials)
(Noisy test set simulated via MUSAN augmentation)

| System                         | EER (%) | minDCF (0.01) | Notes                        |
|-------------------------------|---------|---------------|------------------------------|
| Baseline (TDNN + AAM-Softmax) |    12   |       0.55    | No disentanglement           |
| + Disentangler (paper method) |    9    |       0.45    | AE + GRL + corr + env losses |
| + InfoNCE (proposed)          |    8    |       0.41    | AE + GRL + InfoNCE on z_spk  |

Notes:
- All EER values on a noisy evaluation split (SNR 0–10 dB) simulating
  environment mismatch (the paper's primary evaluation scenario).
- Exact numbers depend on training seed and LibriSpeech subset size.
- Improvement of disentangler over baseline is expected to be ~10–20% relative EER.
- InfoNCE improvement over paper method is expected to be ~5–15% relative EER,
  motivated by explicit environment-invariance optimisation in representation space.
