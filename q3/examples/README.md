# examples/

This directory is populated by `pp_demo.py` with the following files:

| File | Description |
|------|-------------|
| `original.wav` | Source audio (from `--input` arg or synthetic 3-second tone) |
| `converted_male_young.wav` | Converted to Male × Young |
| `converted_male_middle.wav` | Converted to Male × Middle-aged |
| `converted_male_senior.wav` | Converted to Male × Senior |
| `converted_female_young.wav` | Converted to Female × Young |
| `converted_female_middle.wav` | Converted to Female × Middle-aged |
| `converted_female_senior.wav` | Converted to Female × Senior |
| `converted_other_young.wav` | Converted to Other × Young |
| `converted_other_middle.wav` | Converted to Other × Middle-aged |
| `converted_other_senior.wav` | Converted to Other × Senior |
| `fad_results.csv` | Per-conversion FAD scores (generated by `fad_eval.py`) |
| `dnsmos_results.csv` | Per-conversion MOS scores (generated by `dnsmos_proxy.py`) |

## How to regenerate

```bash
# Generate all audio pairs
python pp_demo.py

# Evaluate audio quality
python evaluation_scripts/fad_eval.py \
    --reference examples/original.wav \
    --test_dir  examples/

python evaluation_scripts/dnsmos_proxy.py \
    --dir examples/

# Compare a specific pair
python evaluation_scripts/dnsmos_proxy.py \
    --compare examples/original.wav examples/converted_female_young.wav
```

## Note on model weights

The audio in this directory is generated with **untrained random weights**.
For production-quality conversions, train the model first:

```bash
python train_fair.py --demo --epochs 5
python pp_demo.py --checkpoint checkpoints/checkpoint_final.pt
```
